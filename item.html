Beragam elemen gaya dan kelas CSS untuk dokumentasi

Gaya bersumber dari hasil riset dan untuk mencukupi kebutuhan dalam dokumentasi. Meniru keindahan dari situs-situs ternama, seperti Google.

Headings

Judul halaman paling atas adalah judul halaman. Badan laman tidak boleh mengandung <h1> lainnya, untuk menghindari peramban non-visual yang membingungkan.

Kapan saya harus menggunakan judul?

Code Blocks

Blok Kode, menggunakan PrismJS dan Gaya Okaida.

Tiga Bahasa yang di utamakan, HTML, CSS dan Javascript.

<!-- HTML -->
<figure>
<img class='lazy' src='./e.jpg' data-src='./c.jpg'>
<figcaption>Keterangan: Lazy Loading</figcaption>
</figure>
/*CSS*/
figure{
	display: block;
	margin: 0;
	padding: 27px
}

figure img{
	height: 100%;
	width: 100%
}

figure figcaption{
	font-style: italic;
}
// Javascript
document.addEventListener('DOMContentLoaded', function() {
  var lazyImages = [].slice.call(document.querySelectorAll('img.lazy'));

  if ('IntersectionObserver' in window) {
    let lazyImageObserver = new IntersectionObserver(function(entries, observer) {
      entries.forEach(function(entry) {
        if (entry.isIntersecting) {
          let lazyImage = entry.target;
          lazyImage.src = lazyImage.dataset.src;
          lazyImage.srcset = lazyImage.dataset.srcset;
          lazyImage.classList.remove('lazy');
          lazyImageObserver.unobserve(lazyImage);
        }
      });
    });

    lazyImages.forEach(function(lazyImage) {
      lazyImageObserver.observe(lazyImage);
    });
  } else {
    // Possibly fall back to a more compatible method here
  }
});

Callouts

Catatan: Jenis info ini adalah catatan atau tip biasa.

Perhatian: Jenis info ini menyarankan untuk melanjutkan dengan hati-hati.

Peringatan: Jenis info ini lebih kuat dari Awas; itu berarti 'Jangan lakukan ini.'

Berhasil: Jenis info ini menjelaskan tindakan yang berhasil atau status bebas kesalahan. Hanya digunakan dalam konten interaktif atau dinamis; jangan gunakan di halaman statis biasa.

Poin Kunci: Jenis info ini mendefinisikan konsep penting.

Istilah Utama: Jenis info ini mendefinisikan terminologi penting.

Istilah Utama: Jenis info ini mendefinisikan terminologi penting.

Dogfood: Jenis info ini untuk catatan yang berlaku sementara selama pengujian internal dogfood. Hapus semua info Dogfood sebelum membuat dokumen terlihat oleh publik.
